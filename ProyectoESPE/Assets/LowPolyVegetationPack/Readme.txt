Script LoadScene:
Add all 3 Demo scenes to in the Build settings and in Play mode, press numpad keyboard 0, 1 or 2 to quickly load scenes.

Update (2.1)

Attention!!!! to those who purchased the pack before the update (2.1), it is strongly recommended to make a backup copy of the project. In the new version, the mesh of objects was changed, and the texture extension increased to 256 * 256. Therefore, if you made a replacement for the previous version of the package, then the models from the old version will not be displayed correctly.


Information about the package:

The pack contains 392 fbx models.
Texture Atlas .PSD 256x256;
1 versions of prefabs of vegetation;
46 bonus 3d models;
20 modular terranes;
3 scene demos;
The documentation shows you how to quickly adjust the color palette of the pack.

In this package, you can easily customize the colors of all 3d models. For convenience, all colors were sorted by groups and separate folders. It is enough to open the texture that you want to edit in Photoshop or any other graphics editor that supports the PSD format.

If you have any questions or suggestions, please send them to me by e-mail (darkghost9191@gmail.com).
